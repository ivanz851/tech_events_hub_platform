\section*{Развитие работы}
Разработанная платформа TechEventsHub представляет собой полнофункциональный продукт, но обладает значительным потенциалом для дальнейшего развития. 
Основными направлениями для улучшения платформы являются:

\begin{enumerate}
\item \textbf{Интеграция с новыми источниками анонсов мероприятий.} Планируется реализация дополнительных модулей сбора информации из популярных социальных сетей и мессенджеров (Макс, Яндекс.Мессенджер, ВКонтакте) и со специализированных площадок для организаторов мероприятий с использованием публичных API этих сервисов.

\item \textbf{Внедрение рекомендательной системы.} Накопленную базу данных о предпочтениях пользователей и историю отслеживаемых ими источников можно использовать для обучения моделей машинного обучения. 
Это позволит реализовать интеллектуальную ленту рекомендаций: пользователи смогут видеть новые релевантные источники информации и мероприятия, даже если они не подпадают под строго заданные ими фильтры поиска.

\item \textbf{Реализация функциональности для организаторов мероприятий.} Разработка отдельного личного кабинета для B2B-сегмента, в котором организаторы смогут размещать анонсы своих событий напрямую на платформе TechEventsHub, а также получать аналитику по конверсии из просмотров анонсов в переходы на сайт организатора.

\end{enumerate}
