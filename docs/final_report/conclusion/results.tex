\section*{Результаты}
По итогам работы была успешно спроектирована, разработана и протестирована программная платформа TechEventsHub, предназначенная для автоматизированного поиска, интеллектуального анализа и персонализированной доставки анонсов профильных мероприятий. 
Поставленная цель работы достигнута в полном объеме.

\vspace{1em}

Исходный код разработанной системы размещен в публичном репозитории на платформе GitHub по ссылке: \url{https://github.com/ivanz851/tech_events_hub_platform}.
Инструкция по локальному развертыванию системы приведена в Приложении~\hiddenref{app:local-app-deployment}.

\vspace{1em}

В ходе выполнения выпускной квалификационной работы автором были решены следующие задачи:
\begin{itemize}

    \item Спроектирована и реализована распределенная архитектура серверной части системы, включающая реляционную базу данных для хранения пользовательских настроек и подсистему кэширования для ускорения работы сервисов.

    \item Разработана подсистема автоматизированного сбора информации, извлекающая неструктурированные текстовые данные из открытых каналов в мессенджере Telegram и с веб-страниц, а также реализующая механизм предварительной проверки их доступности.

    \item Интегрирована большая языковая модель YandexGPT, которая позволяет системе автоматически структурировать тексты анонсов, выделяя из них ключевые параметры (дату, место проведения, стоимость и тематику мероприятия).

    \item Обеспечена отказоустойчивость системы посредством реализации шаблонов сетевого взаимодействия: обработки времени ожидания (тайм-аутов), автоматических повторных запросов, ограничения частоты обращений и <<программных предохранителей>> (Circuit Breakers) при сбоях внешних сервисов.

    \item Реализован детерминированный алгоритм фильтрации событий, позволяющий точно сопоставлять извлеченные данные о мероприятиях с заданными персональными предпочтениями каждого пользователя.

    \item Спроектирована подсистема единой идентификации пользователей с поддержкой безопасной авторизации через сервис Яндекс ID и механизмом привязки учетной записи в мессенджере Telegram к основному цифровому профилю.

    \item Реализован маршрутизатор уведомлений, обеспечивающий асинхронную рассылку сформированных дайджестов пользователям по выбранным ими каналам связи (в виде сообщений в Telegram или по электронной почте).

    \item Разработано клиентское веб-приложение с графическим интерфейсом, позволяющее пользователям управлять своими подписками, настраивать параметры поиска и просматривать найденные анонсы в удобном виде.

    \item Настроена программная инфраструктура для мониторинга состояния системы и сбора бизнес-метрик производительности в режиме реального времени.

\end{itemize}

В процессе выполнения работы были приобретены и закреплены навыки:
\begin{itemize}

    \item проектирования распределенных программных систем и построения микросервисной архитектуры с использованием брокера сообщений Apache Kafka;

    \item проектирования реляционных баз данных (PostgreSQL) и написания скриптов миграции схем данных;

    \item обеспечения отказоустойчивости сетевых приложений;

    \item интеграции больших языковых моделей (LLM) в программные продукты для решения задач извлечения данных;

    \item разработки безопасных механизмов авторизации на основе электронных токенов (JWT) и протокола OAuth;

    \item создания современных клиентских веб-интерфейсов на базе компонентного подхода и строгой типизации данных (Next.js, React, TypeScript);

    \item контейнеризации приложений и настройки инструментов для обеспечения наблюдаемости работы программного обеспечения (Prometheus, Grafana).

\end{itemize}
